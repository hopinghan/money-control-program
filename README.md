## 運行規則:
這是一個記帳的程式，一開始會產生可供輸入的選項，來引導輸入，共有3個功能，輸入數字來判斷執行甚麼選項 1新增資料 2.顯示先前紀錄 3.修改紀錄，0.結束程式運行
## 程式用法:
輸入數字1 2 3 0 來判斷輸入選項。
## 程式如何安裝執行
* 在GitHub下載Final project資料夾
* 點擊Download ZIP
* 解壓縮
* 打開exe檔 即可使用
